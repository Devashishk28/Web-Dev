project Name: [#DOM Random Color Box Generator]

This is a simple DOM-based JavaScript project where colorful boxes are generated dynamically when a button is clicked. 
The project demonstrates: 
#DOM manipulation, #event handling, and #dynamic element creation using JavaScript.

## features:

- Generates random colorful boxes on button click
- Each box has a unique RGB color
- Uses DOM manipulation
- Uses addEventListener
- Dynamic element creation using JavaScript

- ## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
- JavaScript (ES6)
- DOM API
- Event Handling

  ## What I Learned:

- How to select elements using querySelector
- How to use addEventListener
- How to create elements dynamically using createElement
- How to generate random RGB colors
- How DOM updates work in real-time
