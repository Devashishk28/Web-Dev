##Project_Name:DOM-DOWNLOADERCARD

A simple Downloader Card UI built using HTML, CSS, and JavaScript DOM Manipulation.
The project simulates a file downloading process with a progress bar and percentage indicator, giving a clean and interactive user experience.

This project was created to practice DOM manipulation, event handling, and UI updates in real time using JavaScript.

##Technologies_Used

HTML5 – Structure
CSS3 – Styling and layout
JavaScript (DOM Manipulation) – Dynamic behaviour

##Features:

Interactive Download Button
Animated Progress Bar
Live Download Percentage
Clean Card-based UI
Smooth button click feedback

##Purpose_of_This_Project:

This project was built to strengthen understanding of:
DOM Selection and Manipulation
Event Listeners
Dynamic UI updates
Progress bar implementation
Basic interactive frontend components
