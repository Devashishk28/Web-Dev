project Name: [## DOM_RandomNameThrower]

A simple JavaScript project built using DOM manipulation.
When the user clicks a button, a random sentence is displayed from a predefined array.

## Features
- Throws random sentence on button click
- Uses JavaScript DOM manipulation
- Beginner-friendly project
- Clean UI

- ## Technologies Used:
- HTML
- CSS
- JavaScript
- DOM Manipulation

- ##  What I Learned
- How to select elements using querySelector
- How to use addEventListener
- How to generate random numbers using Math.random()
- How DOM updates work dynamically
