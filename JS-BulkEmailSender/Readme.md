# 📧 Email Sender Dashboard

A simple and interactive **Email Sender Simulation Dashboard** built using **HTML, CSS, and JavaScript**. This project demonstrates async programming using Promises along with real-time DOM updates.

---

## 🚀 Features

* ✅ Send emails asynchronously (simulated)
* ⚠️ Email validation (invalid emails detection)
* ⏳ Live loader animation while sending
* 📊 Real-time progress counter
* ❌ Retry mechanism for failed emails
* 🎯 Clean and simple UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Promises, DOM Manipulation)

---


## ⚙️ How It Works

1. User enters multiple emails (one per line)
2. Clicks on **Send Emails**
3. Each email:

   * Gets validated
   * Shows loader while processing
   * Displays success or failure
4. Progress counter updates in real-time
5. Failed emails can be retried

---

## 🧠 Concepts Used

* JavaScript Promises
* Async operations (setTimeout simulation)
* DOM Manipulation
* Event Handling
* Array methods (map, forEach)
