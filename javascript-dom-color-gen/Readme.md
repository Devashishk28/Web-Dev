So basically this project was all about DOM.The term 'DOM' stands for
Document Object Model is a programming interface for web documents.
It represents the HTML structure of a webpage as a tree of objects.
Using javascript,We can access,Modify,Add,Or delete HTML elements 
dynamicaaly.

# How it Works?
=>When a web page loads, the browser creates a DOM tree from the HTML.
- Each HTML tag becomes a node.
- JavaScript can access these nodes.
- We can change content, styles, and structure using DOM methods.

  # technologiesused:

- HTML5 – Structure of the webpage
- CSS3 – Styling and layout
- JavaScript (ES6) – Logic and interactivity
- DOM Manipulation – Dynamic content updates
- LocalStorage API – Storing data in browser)

# About This Project

This project focuses on understanding and implementing core DOM manipulation concepts.
It demonstrates how JavaScript interacts with HTML elements to
